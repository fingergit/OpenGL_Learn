#  Lesson1

    本节课创建一个简单的helloworld程序，通过渲染缓冲区，修改View的背景，包括以下内容
    * OpenGL使用的基本流程
    * OpenGL渲染流程

## 项目设置

## OpenGL基本使用流程
    1. 创建并设置OpenGL上下文，后面的渲染操作都在此上下文中进行
    1. 创建帧缓冲区，帧缓冲区是用来最后绘制的缓冲区，由多个缓冲区组成。
       帧缓冲区保存待渲染的数据，渲染目标（如CAEAGLLayer）显示待渲染的数据。
    1. 创建渲染缓冲区，此缓冲区负责颜色的渲染，将此缓冲区设置给上面的帧缓冲区。
    1. 设置渲染缓冲区数据，并渲染。
